## Requirements

- Python >= 3.10
- Network access to a TestBench server

## Install from PyPI

```bash
pip install testbench-cli-reporter
```

## Verify installation

```bash
testbench-cli-reporter --help
```

The package also installs an additional entrypoint alias:

```bash
TestBenchCliReporter --help
```
