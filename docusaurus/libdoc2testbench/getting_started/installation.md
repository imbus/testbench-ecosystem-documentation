# Installation

## Prerequisites

Before installing Libdoc2TestBench, ensure you have:

- **Python 3.9 or higher** installed on your system
- **pip** package manager (usually comes with Python)

:::warning
This extension requires Robot Framework 5.0.0 or later and does not work with earlier versions.
:::

## Install via pip

You can install Libdoc2TestBench via pip using the following command:

```bash
pip install robotframework-libdoc2testbench
```

## Verify Installation

After installation, verify that the tool is correctly installed by checking the version:

```bash
Libdoc2TestBench --version
```

This command will display the Libdoc2TestBench, Robot Framework, and Python versions.
